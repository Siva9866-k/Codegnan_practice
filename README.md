# 🎓 Codegnan Attendance Portal

This is an **individual project** developed using **HTML, CSS, and JavaScript**. It’s a simple and responsive **Attendance Portal** designed for educational or training use.

---

## 🔧 Tech Stack

- HTML
- CSS
- JavaScript

---

## 💡 Features

- ✅ **Live attendance update** — when the checkbox is ticked, the attendance is updated instantly in the UI
- 📱 **Responsive design** — adapts to all screen sizes (desktop, tablet, mobile)
- 🧑‍💻 Clean and user-friendly interface
- 📋 Simple to use and understand — great for demo or prototype scenarios
